Spring MVC
===============================
Bookstore sample project for a Spring 4 MVC .

###1. Technologies used
* Maven 
* Spring MVC

###2. Test
```shell
$ mvn test
```
* Open target/jacoco-ut/index.html from your browser to see test coverage.

###3. To Run this project locally
```shell
$ git clone https://github.com/theritty/bookstore-v2.git
$ mvn jetty:run
```
Access ```http://localhost:8080/store```
